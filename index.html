Quem Somos

Somos uma Fábrica de softwares.

E o que fazemos?

  • Software Engineering

    Possuimos desenvolvedores altamente capacitados para melhorar continuamente suas habilidades ao se perguntarem se há uma maneira melhor de construir alguma coisa. A força motor dessa equipe é encontrar novas formas de entregar valor de negócio para nossos clientes.

  • UX/UI Design

    Uma vez que tenha se usado o UX design, se pode focar nos renamentos visuais da interface de usuário (UI). Isto é, onde os essenciais detalhes cognitivos criam o carácter duradouro da experiência de uma plataforma, como de um dispositivo, software ou aplicativo.

Como podemos atendê-lo?

Escolha o processo que mais se encaixa pra você:

  • Recurso

    Versatilidade

    Escolha prossionais capacitados em diferentes áreas técnicas para desenvolver seu projeto e acompanhe de perto o processo, tendo total liberdade de denir as tarefas para os prossionais contratados.

  • Sprint

    Flexibilidade

    Trabalhando com o apoio da metodologia scrum, desenvolvemos seu projeto através de ciclos semanais ou quinzenais - nomeados como sprints. O projeto é revisado a cada ciclo, adequando-se a novas demandas.

  • Escopo

    Previsibilidade

    Juntos, elaboramos um pré-projeto - documento que apresentará todos os detalhes para seu projeto nal. O projeto será executado dentro da metodologia cascata e ao nal voce receberá o escopo planejado.
